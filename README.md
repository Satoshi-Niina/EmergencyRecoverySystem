ChatGPTへ独自データは、自動整形処理	各チャンクに分割し、Q&A生成＋構造化
データベース（JSON）に追記	append or merge モード（ID or sourceで管理）
fuseは、事前に読み込んだデータをjson/pngに分けて、指定したフォルダに保存
リクエストを受けたfuseが、josnの中てあいまい検索し、紐づけされた,pngをレスポンスする
※ GPTのｊそｎを共有化することで、１つのコンポーネントとフォルダで効率的な検索を構築検討
（ Fuse.js & GPT で検索／回答	チャンクから画像＋説明＋回答を返す構成）

テキストを選択し、下のテキストボックスに転写　→　送信ボタンでテキストをキーワードとしてChatGPT・fuseにリクエスト
フロントサイドの右側エリアにfuseからの検索結果の画像をプレビューする
プレビューされた画像をクリックすることで、元画像＋jsonにある画像に関係するテキストを説明として表示

このプレビュー画像に対して、音声で質問し、テキスト選択することで、ChatGPT・fuseに検索させる
